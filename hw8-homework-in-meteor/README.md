#### Install and Run
1. 安装[Meteor](https://www.meteor.com/) ([Windows Meteor下载 版本：1.1.0.2](https://install.meteor.com/windows))
2. 进入文件夹, 命令行执行：meteor
3. http://localhost:3000/
4. 应用已经部署到meteor.com, 可以通过 http://linmyhw.meteor.com/ 访问
